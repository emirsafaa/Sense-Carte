# Sense-Carte
HORECA işletmelerine yönelik menü optimizasyon ve iş verimliliği artırma platformu.


> **AI-Powered Retail Intelligence & Decision Support Platform**

SenseCarte, işletmelerin satış verilerini anlamlandırmasını, ürün performanslarını analiz etmesini ve veri odaklı stratejik kararlar almasını sağlayan yapay zekâ destekli yeni nesil bir perakende analiz platformudur.

Platform; satış analitiği, kârlılık hesaplamaları, ürün segmentasyonu, stok optimizasyonu, pazar sepeti analizi, müşteri davranış modelleme ve makine öğrenmesi destekli tahminleme süreçlerini tek bir sistem altında birleştirir.

---

# 🎯 Vision

Veriyi yalnızca raporlamak yerine yorumlayan, öneriler sunan ve işletmelerin stratejik karar süreçlerine aktif katkı sağlayan akıllı bir analiz platformu oluşturmak.

SenseCarte, klasik BI araçlarının ötesine geçerek yapay zekâ destekli karar mekanizmaları sunmayı hedeflemektedir.

# 🏗 Architecture

SenseCarte is designed using a modular and scalable architecture.

```
SenseCarte
│
├── Backend (FastAPI)
│      ├── REST API
│      ├── Business Logic
│      ├── Analytics Engine
│      ├── ML Engine
│      ├── Reporting Service
│      └── Database Layer
│
├── PostgreSQL
│
├── Machine Learning
│      ├── Scikit-Learn
│      ├── MLxtend
│      └── NumPy
│
└── Reporting
       ├── PDF Generator
       └── Dashboard
```
